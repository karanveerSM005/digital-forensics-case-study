# Digital Forensics Case Study

A university digital forensics investigation involving the examination of a forensic image from a Samsung Galaxy A3 mobile device.

The project focused on identifying, analysing and interpreting digital artefacts to assess whether they supported or contradicted a set of investigative statements.

> **Academic project:** This repository contains coursework created for educational purposes using a simulated investigation scenario. It is not a real law enforcement case file.

## Overview

The investigation involved analysis of a forensic image rather than the original mobile device, helping preserve evidence integrity during examination.

The report considered:

* Device ownership and user activity
* Messages and communication artefacts
* Browser history and web-search activity
* Application usage
* Image and file artefacts
* Timeline reconstruction
* Cryptocurrency related artefacts
* Concealed or deleted information
* Evidence correlation and reporting

## Forensic Methodology

The investigation was performed using a forensic image of the device rather than the original handset. This helped preserve the source evidence and supported reproducible analysis.

The process included:

1. Reviewing the case background and investigative questions
2. Examining communications, browser artefacts, applications and files
3. Conducting keyword searches to identify relevant evidence
4. Comparing timestamps across artefacts to establish activity sequences
5. Reviewing potentially concealed information and associated applications
6. Correlating evidence across multiple sources
7. Producing evidence-based conclusions for each investigative statement
